## Installing

```shell
dotnet new install Kishotta.AspNetCoreApiTemplate
```

# .Net Web Api Template

An opinionated template for modular .Net Web Apis.

## Usage

```shell
dotnet new kishotta-aspnetcore-api-template -n MyWebApi
```
